## Login - Register

Android Kotlin Login - Register Using MVVM Design Pattern

## Features
- Kotlin
- MVVM Design Pattern
- View Binding
- Kotlin Coroutines
- Android Navigation
- Android Preferences
- Retrofit2 for Client API
- Authentication From Backend API

## Notes
- Build on android studio 4.0
